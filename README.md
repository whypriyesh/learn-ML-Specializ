# Machine Learning Specialization - Andrew Ng

This repository contains my practice implementations and projects for the Machine Learning Specialization by Andrew Ng on Coursera. It serves as a personal learning journal and portfolio of ML concepts.

## Repository Structure

```
├── data/                 # Datasets used in projects
│   └── student_scores.csv # Sample dataset for linear regression
├── src/                  # Source code for ML algorithms and scripts
│   └── linear_regression.py # Implementation of linear regression
├── requirements.txt      # Python dependencies
├── .gitignore           # Files to ignore in version control
└── README.md            # This file
```

## Setup Instructions

1. Clone the repository:
   ```bash
   git clone https://github.com/whypriyesh/learn-ML-Specializ.git
   cd learn-ML-Specializ
   ```

2. Create a virtual environment:
   ```bash
   python -m venv venv
   # On Windows:
   venv\Scripts\activate
   # On macOS/Linux:
   source venv/bin/activate
   ```

3. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
